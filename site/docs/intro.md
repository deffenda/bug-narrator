# BugNarrator Docs

BugNarrator is a local-first narrated software testing tool that helps testers and developers:

- record spoken review sessions
- capture screenshot-backed evidence
- review transcripts and extracted issues
- export durable local artifacts for follow-up work

This docs site now mirrors the canonical onboarding, user, and product-spec pages from the repository while linking back to maintainer-only release and roadmap docs.
