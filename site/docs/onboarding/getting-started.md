# Getting Started

BugNarrator is a local-first narrated testing tool with a production macOS app and an in-progress native Windows workspace.

This page mirrors the canonical onboarding guidance in the repository and points back to the maintainer-only docs when you need more depth.

## If You Are A Tester

1. Download the latest DMG from [GitHub Releases](https://github.com/deffenda/bugnarrator/releases).
2. Install `BugNarrator.app` into `Applications`.
3. Open BugNarrator and add your OpenAI API key in `Settings`.
4. Open `Show Recording Controls` and start a narrated session.
5. Follow the [Tester Narration Guide](https://github.com/deffenda/bugnarrator/blob/main/docs/UserGuide.md#tester-narration-guide) during real testing passes.
